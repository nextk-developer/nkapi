# 소개
NKAI API의 프로토콜은 **REST API(HTTP/JSON)**와 **ZMQ**로 구성됩니다.

### REST API
단방향 요청/응답 형식의 기능에 사용합니다.
- 컴퓨팅 노드
  - 노드 등록
  - 노드 수정
  - 노드 삭제
  - 채널 항목 조회
- 채널
  - 채널 등록
  - 채널 수정
  - 채널 삭제
  - 관심영역 항목 조회
  - 채널 링크 등록 / 해제
- 분석 설정
  - 이벤트 관심영역 등록
  - 이벤트 관심영역 수정
  - 이벤트 관심영역 삭제
  - 분석 제어
    - 시작
    - 종료
    - 초기화
  - 분석 일정 등록
  - 얼굴
    - 얼굴 등록
    - 얼굴 삭제
    - 얼굴 항목 조회
- 레코딩 설정
  - 레코딩 일정 등록
  - 저장된 데이터 날짜 조회
- 플레이백
  - 플레이백 요청
- 메타 데이터
  - 메타 데이터 조회
- 통계 데이터 조회
- 시스템
  - 시스템 로그 조회
  - 시스템 명령 전달
- 모델 설정

- 공통 항목
